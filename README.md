Dalmoa (달모아) - Android App
구독 관리 서비스 '달모아'의 공식 안드로이드 어플리케이션입니다.
흩어져 있는 구독 서비스를 한눈에 확인하고, 매달 지출되는 고정 비용을 스마트하게 관리하세요.

---

🔗 관련 저장소
* Backend API: Dalmoa_Backend 저장소 링크 (https://github.com/...)

---

🛠 Tech Stack  * Language: Kotlin
 * Architecture: MVVM + Clean Architecture (Base)
 * UI: XML, ViewBinding, Material Design 3
 * Networking: Retrofit2, OkHttp3
 * Navigation: Jetpack Navigation Component
 * Async: Kotlin Coroutines

---

✨ Key Features 
* Dashboard: 전체 구독 현황 및 월간 지출 통계 시각화
* Subscription Feed: 등록된 구독 서비스 리스트 관리
* Auth: 회원가입 및 로그인 (JWT 기반)
* Auto-Calculation: 해외 결제 구독 서비스의 실시간 환율 계산 반영

---

🚀 Getting Started

Prerequisites
* Android Studio Ladybug 이상 권장
* JDK 21
* Minimum SDK: 24 (Android 7.0)

Configuration
local.properties 파일에 필요한 설정을 추가하세요 (필요 시).

Build & Run
Android Studio에서 프로젝트를 열고 app 모듈을 실행하세요.
